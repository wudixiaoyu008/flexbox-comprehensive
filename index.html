flex-direction

row || column || row-reverse || column-reverse

flex-wrap

default: nowrap || wrap || wrap-reverse

flex-flow

flex-flow is shorthand property which takes flex-direction and flex wrap

justify-content

flex-start || flex-end || center || space-between || space-around

align-items

flex-start || flex-end || center || stretch || baseline

default: stretch, so they fill the entire height of the flex container

align-content

default: stretch || flex-start || flex-end || center

mainly used on multi-line flex-containers

flex-grow || flex-shrink

The "flex-grow" and "flex-shrink" properties control how much a flex-item should “grow” (extend) if there are extra spaces, or “shrink” if there are no extra spaces.

They may take up any values ranging from 0 to any positive number. 0 || positive number

flex-grow, be default is 0, you can change it to 1, it will occupy all available space, the switch is turn on

at this time, if resize the browser, the flex item will shrink to accomodate the new screen width

since by default, the flex-shrink is turned on, is 1, you can change it to 0, so that to turn off

flex-basis

The flex-basis property specifies the initial size of a flex-item. Before the flex-grow or flex-shrink properties adjust it's size to fit the container or not.

The default value is flex-basis: auto. Flex-basis can take on any values you'd use on the normal width property. That is, percentages || ems || rems || pixels etc

flex shorthand

flex: 0 1 auto; flex-grow || flex-shrink || flex-basis
